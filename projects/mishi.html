MISHI

View on GitHub View Demo

Overview

MISHI is a full-stack web application designed to help track and foster stray cats. Built with React and Firebase, it provides a comprehensive platform for cat lovers and rescue organizations to collaborate on helping stray animals find homes.

The application integrates Google Maps API for location tracking, allowing users to pin exact locations where stray cats are spotted. With role-based authentication, different user types can access features appropriate to their needs—from casual observers to active rescue coordinators.

Project Vision

MISHI aims to create a centralized, community-driven ecosystem for stray cat rescue and adoption. The vision is to connect compassionate individuals, rescue organizations, and potential adopters through technology, making it easier to track, care for, and find homes for stray cats in need.

By providing real-time location data, resource coordination, and seamless communication tools, MISHI empowers communities to take collective action in addressing the stray cat population humanely and effectively.

Technology Stack

React Firebase Firestore Google Maps API OAuth 2.0 JavaScript CSS3

Current Status

MISHI is functionally complete and serves its core purpose of tracking and fostering stray cats. While I'm currently focused on other projects, I may revisit MISHI in the future to add new features or improvements as needed. The codebase is stable and the application is fully operational.

Key Features

  • Interactive Map Integration: Google Maps API for precise location pinning and visualization of stray cat sightings
  • Role-Based Authentication: OAuth-enabled account management with different permission levels
  • Real-Time Updates: Firebase Firestore for instant data synchronization across all users
  • Resource Sharing: Community-driven platform for sharing food, shelter, and adoption resources
  • Secure Data Management: Firestore security rules ensuring data privacy and integrity
  • Responsive Design: Works seamlessly across desktop and mobile devices

Development Process

MISHI started as a passion project to address a real community need. The development process involved:

  • Architecting a scalable database structure in Firestore
  • Implementing secure authentication flows with OAuth
  • Integrating and optimizing Google Maps API for performance
  • Designing an intuitive user interface for both technical and non-technical users
  • Writing comprehensive Firestore security rules to protect user data
  • Continuous iteration based on user feedback and testing

Project Significance

MISHI represents my first major full-stack project combining real-time data synchronization, geolocation services, and role-based authentication. It taught me valuable lessons about working with cloud platforms, designing scalable database architectures, and building applications that serve real community needs.

Beyond the technical skills, MISHI demonstrated the power of technology to create positive social impact. Seeing the platform help connect rescuers with stray cats in need reinforced my belief that software development can be a force for good in the world.
